# Resolver

Read this file before creating a new page.

## Decision tree

1. Is it a specific named human? → `people/`
2. Is it an organization, company, institution, or brand? → `companies/`
3. Is it a financing event, transaction, or terms discussion? → `deals/`
4. Is it a meeting record, transcript, or meeting synthesis? → `meetings/`
5. Is it something actively being built or operated with concrete execution? → `projects/`
6. Is it a possible thing that is not yet being actively built? → `ideas/`
7. Is it a teachable framework, mental model, or concept? → `concepts/`
8. Is it developed prose, an essay, or a draft meant to be read as writing? → `writing/`
9. Is it a major long-running workstream or life program? → `programs/`
10. Is it strategy or operations for an institution or organization I run or help run? → `org/`
11. Is it politics, government, public policy, or civic landscape? → `civic/`
12. Is it media production, publishing, public narrative, or content ops? → `media/`
13. Is it private reflection, personal health, or interior life? → `personal/`
14. Is it household logistics, properties, or domestic operations? → `household/`
15. Is it hiring pipeline, candidate evaluation, or recruiting process? → `hiring/`
16. Is it a raw import, archived source file, or supporting material? → `sources/`
17. Is it a reusable prompt or prompt pattern? → `prompts/`
18. Is it a task list, routine, checklist, or operating note? → `ops/`
19. If nothing fits cleanly yet → `inbox/`

## Tie-break rules

- Concept vs idea: if it is teachable, it is a concept; if it is buildable, it is an idea.
- Idea vs project: if there is active execution, it is a project.
- Personal vs concept: if it is private and autobiographical, it is personal.
- Writing vs concept: writing is developed prose; concepts are distilled reference pages.
- Sources vs page: raw artifacts go in `sources/`; synthesized knowledge gets its own page elsewhere.

## Filing rule

When in doubt, prefer a single primary home and cross-link rather than duplicating pages.

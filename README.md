# second brain

Personal knowledge brain powered by GBrain.

This repo is the human-readable markdown layer. GBrain provides indexing, search,
enrichment, sync, and agent workflows on top of it.

## v1 stance

This repo starts with the GBrain recommended schema almost exactly as written,
including a resolver-first filing model, compiled-truth-plus-timeline page design,
and a priority-based task system in `ops/tasks.md`.

## Core operating rules

1. Read `RESOLVER.md` before creating a new page.
2. Use one primary home per page.
3. Keep synthesis above the line and evidence/timeline below the line.
4. Put operational tasks in `ops/tasks.md`.
5. Use `inbox/` for captures that are not yet classified.

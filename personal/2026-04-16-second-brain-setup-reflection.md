## State
- Today was more about learning than shipping; William described it as a day with "not many deliverables" but still "a very great day" because of how much he learned. [Source: User, voice note, 2026-04-16]
- `second-brain` setup with GBrain is now far enough along that it feels real and usable, even if the overall local folder layout still feels like it is evolving. [Source: User, voice note, 2026-04-16]
- Hermes already restores something William missed: the ability to talk out loud while commuting, brainstorm, reflect, and "breathe again." [Source: User, voice note, 2026-04-16]
- Claude Code's official integrations currently feel more polished and reliable than many third-party plugins. Hermes still feels attractive because MCPs and skills are comparatively easy to make, but support quality is uneven. [Source: User, voice note, 2026-04-16]
- The hardest integration today was Slack. The current path still feels app-mediated, while William mainly wants Slack connected as himself so the agent can see what he can see and act in a more natural way. [Source: User, voice note, 2026-04-16]
- Using Hermes through the OpenAI subscription felt weak for agentic tool use. Raising reasoning helped, but William still felt Claude was much stronger at knowing when to use tools correctly and how to act on instructions instead of misfiling them. [Source: User, voice note, 2026-04-16]
- TTS experimentation was a major frustration today: the voices were not good, instructions were not being followed, and the overall experience felt like a nightmare rather than a helpful workflow. [Source: User, voice note, 2026-04-16]
- The current stance is pragmatic: keep using vanilla Hermes for now, go step by step, and avoid overcomplicating the stack too early. [Source: User, voice note, 2026-04-16]
